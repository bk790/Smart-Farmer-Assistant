# 🌾 Smart Farmer Assistant

A multilingual Smart Farmer Assistant application designed to help rural farmers make informed agricultural decisions through crop recommendations, fertilizer estimation, profit analysis, and record management.

## 📌 Project Overview

Smart Farmer Assistant is an Innovation and Design Thinking project aimed at supporting farmers with easy-to-use digital tools. The application provides agricultural guidance in multiple languages and stores farmer records for future reference.

## ✨ Features

### 🌍 Multilingual Support

* English
* Hindi (हिन्दी)
* Kannada (ಕನ್ನಡ)

### 🌾 Crop Recommendation

Recommends suitable crops based on:

* Soil Type
* Season
* Water Availability

### 🧪 Fertilizer Calculator

Calculates approximate fertilizer requirements based on land area.

### 💰 Profit Estimator

Estimates:

* Expected Revenue
* Expected Profit

### 📊 Farmer Records

* Saves farmer information
* Stores crop recommendations
* Maintains profit analysis records
* Exports records as CSV

### 🔄 Automatic Translation

Translator for dynamic interface translation with agricultural terminology support.

## 📥 Installation

Clone or download the project.

Install required packages:

```bash
pip install streamlit pandas deep-translator
```

---

## ▶️ Running the Application

```bash
streamlit run app.py
```

The application will automatically open in your browser.

## 📖 How to Use

### Crop Recommendation

1. Select language.
2. Enter farmer details.
3. Select soil type.
4. Select season.
5. Select water availability.
6. Click **Recommend Crop**.

### Fertilizer Calculator

1. Enter land area in acres.
2. Click **Calculate Fertilizer**.
3. View fertilizer requirement.

### Profit Estimator

1. Enter cultivation cost.
2. Enter expected yield.
3. Enter market price.
4. Click **Estimate Profit**.
5. View revenue and profit.

### Farmer Records

1. Open the Farmer Records section.
2. View saved records.
3. Download records as CSV.
